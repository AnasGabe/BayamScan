╔══════════════════════════════════════════════════════════════════════════════╗
║  💡 KEY METRICS EXPLAINED                                                     ║
╚══════════════════════════════════════════════════════════════════════════════╝

Accuracy (Akurasi)
  ├─ Formula: Correct predictions / Total predictions
  ├─ Range: 0-100% (higher is better)
  ├─ Target: > 95%
  └─ Note: Can be misleading with imbalanced data

Recall (Sensitifitas)  
  ├─ Formula: True positives / All actual positives
  ├─ Question: "Of all actual BUSUK, how many did we correctly identify?"
  ├─ Range: 0-100% (higher is better)
  ├─ Target: Each class > 90%
  └─ Note: Important for missing diagnoses

Precision (Presisi)
  ├─ Formula: True positives / All predicted positives
  ├─ Question: "Of all we predicted as BUSUK, how many were actually BUSUK?"
  ├─ Range: 0-100% (higher is better)
  ├─ Target: Each class > 90%
  └─ Note: Important for false positives

F1-Score
  ├─ Formula: 2 × (Precision × Recall) / (Precision + Recall)
  ├─ Range: 0-1 (1.0 is perfect, higher is better)
  ├─ Target: Each class > 0.85
  └─ Note: Balanced measure of precision & recall

Overfitting Gap
  ├─ Formula: Training Accuracy - Test Accuracy
  ├─ Question: "Is model just memorizing training data?"
  ├─ Target: < 2% (good), < 5% (acceptable)
  └─ Note: Indicates generalization ability

Class Balance (Recall Diff)
  ├─ Formula: Max(all recalls) - Min(all recalls)
  ├─ Question: "Is model fair to all classes?"
  ├─ Target: < 10% (very balanced)
  ├─ Your model: 3.3% ✅
  └─ Note: Detects bias to specific class
